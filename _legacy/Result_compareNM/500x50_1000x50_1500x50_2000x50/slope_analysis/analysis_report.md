# 相变前斜率分析报告

## 数据概述

本次分析包含 **4** 组不同的 (N, M) 配置：

- N=500, M=50 (N/M=10.0): 斜率 = 0.1567, 相变点 α_c ≈ 2.30
- N=1000, M=50 (N/M=20.0): 斜率 = 0.0829, 相变点 α_c ≈ 2.60
- N=1500, M=50 (N/M=30.0): 斜率 = 0.0548, 相变点 α_c ≈ 2.80
- N=2000, M=50 (N/M=40.0): 斜率 = 0.0411, 相变点 α_c ≈ 2.80

## 核心发现

### 过原点线性拟合 (最重要)

**公式**: `slope = 1.5921 × (M/N)`

**R² = 0.9994**

拟合效果优秀，数据强烈支持 **斜率与 M/N 成正比** 的假设。

### 幂律拟合

**公式**: `slope = 1.3990 × (M/N)^0.950`

**R² = 0.9995**

幂律指数 **0.950 ≈ 1**，与线性关系一致，验证了 slope ∝ M/N 的假设。

## 物理意义

相变前线性增长区域的斜率反映了系统在低观测密度下的学习效率：

- **斜率 ∝ M/N** 意味着：当隐变量维度 M 相对于观测维度 N 越大时，
  每增加一个单位的观测密度 α，Q_Y 增长越快
- 这与直觉一致：M/N 越大，信息冗余度越高，更容易从部分观测中恢复信号

## 结论

数据支持以下经验公式：

```
dQ_Y/dα ≈ 1.59 × (M/N)    (相变前线性区)
```

即相变前 Q_Y 的增长速率与 M/N 成正比，比例系数约为 1.59。
