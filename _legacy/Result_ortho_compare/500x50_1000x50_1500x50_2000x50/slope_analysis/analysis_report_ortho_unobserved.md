# 相变前斜率分析报告 - Q_Y_unobserved (Orthogonal)

## 数据概述

本次分析包含 **4** 组不同的 (N, M) 配置：

- N=500, M=50 (N/M=10.0): 斜率 = 0.0954, 相变点 α_c ≈ 2.40
- N=1000, M=50 (N/M=20.0): 斜率 = 0.0550, 相变点 α_c ≈ 2.60
- N=1500, M=50 (N/M=30.0): 斜率 = 0.0387, 相变点 α_c ≈ 2.80
- N=2000, M=50 (N/M=40.0): 斜率 = 0.0306, 相变点 α_c ≈ 2.90

## 核心发现

### 过原点线性拟合

**公式**: `slope = 1.0076 × (M/N)`

**R² = 0.9928**

拟合效果优秀，数据强烈支持 **斜率与 M/N 成正比** 的假设。

## 物理意义

**正交教师模型**通过 QR 分解强制 W^T W = I_M，消除了有限尺寸涨落。

- 标准教师的 Gram 矩阵 G = (M/N) W^T W = I + Δ，其中 Δ 是 O(1/√N) 的随机涨落
- 正交教师强制 Δ = 0，因此低 α 区域的线性偏移应该更小

如果正交教师的斜率系数显著小于标准教师，说明 **有限尺寸涨落是导致低 α 区域 Q_Y 偏高的主要原因**。

## 结论

数据支持以下经验公式：

```
dQ_Y_unobs/dα ≈ 1.01 × (M/N)    (相变前线性区)
```

即相变前 Q_Y_unobserved (Orthogonal) 的增长速率与 M/N 成正比，比例系数约为 **1.01**。
