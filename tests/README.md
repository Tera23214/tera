# Tests Directory

This is a **temporary test directory** for development and debugging.

## Purpose

- Store temporary test scripts
- Run quick experiments
- Debug code before integrating into main modules

## Guidelines

1. Files here are **not tracked by Git**
2. Clean up after testing is complete
3. Do not store important data here

## Usage

```bash
# Run a test script
python tests/my_test.py

# Clean up
rm -rf tests/*.py
```

---

*This directory is excluded from version control.*
