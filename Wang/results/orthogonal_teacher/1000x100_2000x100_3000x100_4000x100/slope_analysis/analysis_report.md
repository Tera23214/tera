# 相变前斜率分析报告 - Q_Y (Standard)

## 数据概述

本次分析包含 **4** 组不同的 (N, M) 配置：

- N=1000, M=100 (N/M=10.0): 斜率 = 0.1689, 相变点 α_c ≈ 2.30
- N=2000, M=100 (N/M=20.0): 斜率 = 0.0891, 相变点 α_c ≈ 2.60
- N=3000, M=100 (N/M=30.0): 斜率 = 0.0582, 相变点 α_c ≈ 2.70
- N=4000, M=100 (N/M=40.0): 斜率 = 0.0482, 相变点 α_c ≈ 2.80

## 核心发现

### 拟合模型对比

| 模型 | 公式 | R² |
|------|------|-----|
| 过原点线性 | `slope = 1.719838 * (M/N)` | 0.9989 |
| 过原点二次 | `slope = -1.8502×(M/N)² + 1.8729×(M/N)` | 0.9997 |
| 幂律 | `slope = 1.447023 * (M/N)^0.9330` | 0.9989 |
| 平方根 | `slope = 0.4395×√(M/N)` | 0.9545 |
| 带截距线性 | `slope = 1.627251 * (M/N) + 0.006327` | 0.9990 |
| 二次多项式 | `slope = -0.3740×(M/N)² + 1.6754×(M/N) + 0.0051` | 0.9990 |

**最佳拟合模型**: 过原点二次 (R² = 0.9997)

### 过原点线性拟合

**公式**: `slope = 1.7198 × (M/N)`

**R² = 0.9989**

拟合效果优秀，数据强烈支持 **斜率与 M/N 成正比** 的假设。

## 物理意义

相变前线性增长区域的斜率反映了系统在低观测密度下的学习效率：

- **斜率 ∝ M/N** 意味着：当隐变量维度 M 相对于观测维度 N 越大时，
  每增加一个单位的观测密度 α，Q_Y 增长越快
- 这与直觉一致：M/N 越大，信息冗余度越高，更容易从部分观测中恢复信号

## 结论

数据支持以下经验公式：

```
dQ_Y/dα ≈ 1.72 × (M/N)    (相变前线性区)
```

即相变前 Q_Y (Standard) 的增长速率与 M/N 成正比，比例系数约为 **1.72**。
