# 相变前斜率分析报告 - Q_Y_unobserved (Orthogonal)

## 数据概述

本次分析包含 **4** 组不同的 (N, M) 配置：

- N=1000, M=100 (N/M=10.0): 斜率 = 0.1128, 相变点 α_c ≈ 2.30
- N=2000, M=100 (N/M=20.0): 斜率 = 0.0717, 相变点 α_c ≈ 2.60
- N=3000, M=100 (N/M=30.0): 斜率 = 0.0488, 相变点 α_c ≈ 2.80
- N=4000, M=100 (N/M=40.0): 斜率 = 0.0394, 相变点 α_c ≈ 2.80

## 核心发现

### 拟合模型对比

| 模型 | 公式 | R² |
|------|------|-----|
| 过原点线性 | `slope = 1.227324 * (M/N)` | 0.9842 |
| 过原点二次 | `slope = -5.7063×(M/N)² + 1.6994×(M/N)` | 0.9998 |
| 幂律 | `slope = 0.624315 * (M/N)^0.7397` | 0.9942 |
| 平方根 | `slope = 0.3208×√(M/N)` | 0.9838 |
| 带截距线性 | `slope = 0.966626 * (M/N) + 0.017814` | 0.9868 |
| 二次多项式 | `slope = -6.2139×(M/N)² + 1.7673×(M/N) + -0.0018` | 0.9990 |

**最佳拟合模型**: 过原点二次 (R² = 0.9998)

### 过原点线性拟合

**公式**: `slope = 1.2273 × (M/N)`

**R² = 0.9842**

拟合效果优秀，数据强烈支持 **斜率与 M/N 成正比** 的假设。

## 物理意义

**正交教师模型**通过 QR 分解强制 W^T W = I_M，消除了有限尺寸涨落。

- 标准教师的 Gram 矩阵 G = (M/N) W^T W = I + Δ，其中 Δ 是 O(1/√N) 的随机涨落
- 正交教师强制 Δ = 0，因此低 α 区域的线性偏移应该更小

如果正交教师的斜率系数显著小于标准教师，说明 **有限尺寸涨落是导致低 α 区域 Q_Y 偏高的主要原因**。

## 结论

数据支持以下经验公式：

```
dQ_Y_unobs/dα ≈ 1.23 × (M/N)    (相变前线性区)
```

即相变前 Q_Y_unobserved (Orthogonal) 的增长速率与 M/N 成正比，比例系数约为 **1.23**。
